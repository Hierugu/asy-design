# Описание дизайна сайта «ДОМОСТРОЙ» для реализации на Next.js

## Общая структура

**Сайт:** Интернет-каталог строительных материалов и услуг с акцентом на российских производителей.

**Навигация:**
- Логотип: «ДОМОСТРОЙ»
- Горизонтальное меню: Главная, Каталог, О нас, Контакты, Заказать проект

**Основные цветовые акценты:**
- Чистый, минималистичный дизайн
- Четкие разделы с заголовками
- Возможное использование синих/зеленых акцентов для кнопок и выделения

## Технические требования для реализации

**Фреймворк:** Next.js 14+ (App Router)
**Стили:** Tailwind CSS
**Компоненты:**
- `Header` с логотипом и навигацией
- `HeroSection` для главной страницы
- `TargetAudience` с тремя колонками
- `AboutSection` для страницы «О нас»
- `Footer` с контактами и ссылками
- Кнопки с призывом к действию («Заказать проект»)

**Адаптивность:**
- Мобильная, планшетная и десктопная версии
- Три колонки на десктопе → одна колонка на мобильном

**Шрифты:**
- Чистый, современный шрифт без засечек (например, Inter, Roboto)

**Иконки:**
- React Icons или аналогичная библиотека для иконок в блоках
